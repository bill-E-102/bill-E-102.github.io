Grassmannian Codes from Stratified Frames

AFIT Applied Mathematics Master’s Thesis

#frame_theory

Developed a general method to construct equichordal tight fusion frames (ECTFFs)—optimal Grassmannian codes—using stratified unit norm tight frames. This unifies known constructions between group difference families and resolvable balanced incomplete block designs, further yielding infinitely many new ECTFFs.

Two-Channel Correlator Receiver Simulation

EENG 669 Digital Communications I Project 2

#digital_communications #signal_processing

Implementing two-channel correlator receivers with orthogonal BFSK modulation using ideal and Butterworth-filtered detection; further comparing receivers based off effective signal-to-noise ratio (SNR) and bit error rates (BER) via Monte Carlo simulation

Learning With Errors via Integer Programming

OPER 613 Integer Programming Final Project

#cryptography #mathematical_optimization

Implementing an attack on the discrete version of the search learning with errors problem via a reduction to and formulation of the integer programming problem in Pyomo. Recovers the secret vector by minimizing a constrained $\ell_1$-norm, comparing the results against lattice reduction methods for benchmarking.

UVA Landmark Recognition with CNNs

CS 4774 Machine Learning Codeathon 2

#machine-learning

Developed two image classification models using TensorFlow, Keras, NumPy, and scikit-learn to identify 18 UVA campus buildings from a custom-labeled dataset. Created a custom CNN model and implemented a transfer learning model (ResNet101), comparing performance off accuracy and training time.
